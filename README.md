# AI Travel Planner (Streamlit + Gemini + SerpAPI)

https://ai-travel-planner-bqf2xxeq5yggt3ufuunmkg.streamlit.app/

An intelligent travel planning web application that generates personalized itineraries using AI, live flight data, and destination insights.

---

## Features

- 🌍 AI-powered travel itinerary generation
- ✈️ Live flight search using Google Flights (SerpAPI)
- 🏨 Hotel & restaurant recommendations
- 🧠 AI research using Gemini model
- 🗺️ Personalized trip planning based on:
  - Budget
  - Travel theme
  - Duration
  - Activities
- 🎯 Smart itinerary optimization
- 💻 Interactive Streamlit UI

---

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- SerpAPI (Google Flights)
- Agno AI Agents
- Pandas / JSON
- HTML/CSS (custom UI styling)
